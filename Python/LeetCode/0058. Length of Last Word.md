计算一句话中最后一个单词的长度


```python
class Solution(object):
    def lengthOfLastWord(self, s):
        """
        :types: str
        :rtype: int
        """
        if not s:
            return 0
        s = s.rstrip() # 如果最右存在空格，则会导致结果错误，所以先对字符串进行处理，去除最右空格
        count = 0
        while len(s) > 0 and s[-1] != " ":
            s = s[:-1]
            count += 1
        return count
