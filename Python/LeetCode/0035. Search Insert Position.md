给定一个有序array， 以及一个target， 如果target与array 中的某个数字相等，则返回该数字的位置，如果不是，则返回应该将target 插入的位置


```python
class Solution(object):
    def searchInsert(self, nums, target):
        """
        :typenums: List[int]
        :typetarget: int
        :rtype: int
        """
        if target > nums[-1]:
            return len(nums)
        if target < nums[0]:
            return 0
        for i in range(1,len(nums)):
            if target == nums[i]:
                return i
            if target < nums[i] and target > nums[i-1]:
                return i
```
