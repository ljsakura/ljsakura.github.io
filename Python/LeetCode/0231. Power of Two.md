Given an integer, write a function to determine if it is a power of two. 

给定一个整数，判断其是否是2的幂次
```python
class Solution(object):
    def isPowerOfTwo(self, n):
        """
        :typen: int
        :rtype: bool
        """
        if n <= 0: #需要考虑小于零的情况
            return False
        while n > 1:
            if n % 2 != 0:
                return False
            n /= 2
        return True
